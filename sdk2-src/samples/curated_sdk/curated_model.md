## Model for CLI

```bash
az ml model create --name <model-name> --version <version> --path <path>
az ml model create --file <file_path>
az ml model show --name <model-name> --version <version>
az ml model list
az ml model list --name <model-name>
az ml model create --file <file_path> --set name=<some-name>
az ml model update --file <file_path>
az ml model delete --name <model-name> --version <version>
```
